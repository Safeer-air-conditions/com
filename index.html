البيانات التي نجمعها

  • الاسم ورقم الهاتف عند التواصل عبر واتساب أو الاتصال المباشر.
  • عنوان الموقع (الحي/المدينة) لتحديد إمكانية تنفيذ الخدمة والمعاينة.
  • أي تفاصيل يقدمها العميل طواعية بخصوص نوع الجهاز أو طبيعة الطلب.

كيف نستخدم هذه البيانات

  • التواصل معك لتنسيق موعد المعاينة أو التنفيذ.
  • تقديم عرض سعر أو تقييم للجهاز المستعمل.
  • تحسين جودة الخدمة المقدمة لك مستقبلًا.

مشاركة البيانات

لا تقوم "سفير لخدمات التكييف" ببيع أو مشاركة بياناتك مع أي طرف ثالث لأغراض تسويقية دون موافقتك الصريحة. قد تتم مشاركة الحد الأدنى من المعلومات مع الفني المكلف بالزيارة فقط لغرض تنفيذ الخدمة.
